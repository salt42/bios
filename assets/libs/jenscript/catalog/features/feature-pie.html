Pie Feature
 Draw pies with labels and effects.
Key Description
Pie Plugin Pie plugin takes responsibility to paint pies
Pie Pie is defined by it's center and radius
Pie Slice Pie Slice has to be registered in pie with value and theme color
Stroke,Fill, Effect Create beautiful pie with stroke, fill and effects.
Labels Put labels on slice with bounded, radial, border or path labels.
Listener Allow pie action listening and create animation with effect, label or user intents.
