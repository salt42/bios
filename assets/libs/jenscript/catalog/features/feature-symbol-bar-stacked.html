Symbol Bar Stacked Feature
 Draw Bar, Stacked Bar, Point and Curve symbols with labels and effects.
Key feature
Symbol Plugin Symbol plugin holds Symbol objects
Symbol Symbol is unidimensional (x or y dimension). One dimension is projected, the other is lay out as a symbol in component pixel definitions.
Symbol type Bar, Stacked Bar, Point and curve symbol
Vector dimension Base and/or value, this part of symbol is projected on x or y dimension
Symbol Filler Symbol is lay out with strut or glue relative to bound and other symbols.
Rendering Create beautiful symbols with strokes, fills, effects, icons and labels
Listener Interact with symbol by listener
Animation Create beautiful symbols animations
