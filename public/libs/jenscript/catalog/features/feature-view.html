View Feature
 View defines the chart component.it has 5 children components, outer parts and device part.
Feature Description
View View is user end chart, view hosts projections.
View projections View holds user projections coordinate systems.
View backgrounds Translucent, colorized, textured or image view background.
View parts View has 5 parts components, North, South, East and West outers parts and Device part.
