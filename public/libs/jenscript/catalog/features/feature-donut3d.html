Donut3D Feature
 Draw donut3D with labels and effects.
Key Description
Donut3D Plugin Donut3D plugin takes responsibility to paint donuts
Donut3D Donut3D is defined by it's center, inner and outer radius, tilt and thickness.
Donut3D Slice Donut3D Slice has to be registered in donut with value and theme color
Paint Create beautiful donut with paint renderer.
Labels Put labels on slice with bounded, radial, border or path labels.
Listener Allow donut action listening and create animation with effect, label or user intents.
