Donut2D Feature
 Draw donut2D with labels and effects.
Key Description
Donut2D Plugin Donut2D plugin takes responsibility to paint donuts
Donut2D Donut2D is defined by it's center and inner and outer radius
Donut2D Slice Donut2D Slice has to be registered in donut with value and theme color
Stroke,Fill, Effect Create beautiful donut with stroke, fill and effects.
Labels Put labels on slice with bounded, radial, border or path labels.
Listener Allow donut action listening and create animation with effect, label or user intents.
